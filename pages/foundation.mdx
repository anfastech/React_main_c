#### Writing Basic C Program

To start programming in C , you need to understand the structure of a basic C program.<br /> Hereâ€™s a simple example:

```c filename="main.c" /<conio.h>/ /clrscr();/ /getch();/ copy 
#include <stdio.h>
#include <conio.h>
 
void main() {
    int number = 20 ;
    clrscr();

    printf("the number = %d",number);
    getch();
}

```

#### Explanation

`#include <stdio.h>`: This line tells the compiler to include the Standard Input Output library before compiling the program. This library is necessary for functions like printf. <br />
`void main()`: This is the main function where the execution of the program begins. Every C program must have a main function. <br />
`printf("Value = %g",value);`: This function prints the text to the screen. <br /> The `%g` is a datatype representation characters that print the representing data in the console.
