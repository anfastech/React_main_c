# Scientific Calculator

This C program implements a basic scientific calculator using the console. The calculator offers various mathematical functions including trigonometric functions (sin, cos, tan), logarithmic functions (natural log, base-10 log), exponentiation (e^x), square root, and factorial calculation. 
The user is presented with a menu to choose the desired operation. Depending on the choice, the program performs the corresponding calculation using the input value and displays the result. The program continues to run until the user chooses to exit by selecting the appropriate option from the menu.

### Code Breakdown

## Algorithm

1. **Start**
2. **Initialize Variables**
   - `float x;`
   - `int choice;`
   - `long f;`
3. **Display Menu**
   - Show available operations.
4. **Accept User Choice**
   - Read user input for the choice.
5. **Check Choice Validity**
   - If choice is valid (1-9), proceed.
   - If choice is 9, exit the program.
6. **Prompt for Value**
   - Read the value of `x` from the user.
7. **Perform Calculation Based on Choice**
   - Use `switch` statement to execute the selected operation:
     - **1:** Calculate `sin(x)` in degrees.
     - **2:** Calculate `cos(x)` in degrees.
     - **3:** Calculate `tan(x)` in degrees.
     - **4:** Calculate `log(x)` (natural logarithm).
     - **5:** Calculate `log10(x)` (base-10 logarithm).
     - **6:** Calculate `e^x`.
     - **7:** Calculate `sqrt(x)`.
     - **8:** Calculate factorial of `x`.
8. **Display Result**
   - Print the result of the calculation.
9. **Repeat**
   - Go back to the menu for another operation.
10. **End**

```c filename="Detailed" copy showLineNumbers
#include <stdio.h>  // Header file for input and output functions
#include <conio.h>  // Header file for console input/output functions
#include <math.h>   // Header file for mathematical functions

#define PI (22.0 / 7)  // Define a constant PI with an approximation of 22/7

int main() {
    float x;           // Variable to store the user input value for calculations
    int choice;        // Variable to store the user's menu choice
    long f;            // Variable to store the result of factorial calculation
    int i;             // Loop variable

    do {
        clrscr();      // Clear the console screen

        // Display the calculator menu
        printf("\nScientific Calculator");
        printf("\n---------------------");
        printf("\n\t1. Sin(x)");       // Option to calculate sine of x
        printf("\n\t2. Cos(x)");       // Option to calculate cosine of x
        printf("\n\t3. Tan(x)");       // Option to calculate tangent of x
        printf("\n\t4. Logn(x)");      // Option to calculate natural logarithm of x
        printf("\n\t5. Log10(x)");     // Option to calculate base-10 logarithm of x
        printf("\n\t6. e^x");          // Option to calculate e raised to the power x
        printf("\n\t7. Squareroot(x)"); // Option to calculate square root of x
        printf("\n\t8. x!");           // Option to calculate factorial of x
        printf("\n\t9. Exit");         // Option to exit the program
        printf("\n\nEnter your choice (1-9): ");
        scanf("%d", &choice); // Read the user's choice

        if (choice < 1 || choice > 9) continue; // Skip invalid choices
        else if (choice == 9) break;           // Exit the loop if choice is 9

        printf("\nx? ");
        scanf("%f", &x); // Read the value of x

        // Perform the selected operation
        switch (choice) {
            case 1: // Calculate sin(x)
                printf("\n%g", sin(x * PI / 180)); // Convert x to radians and compute sin(x)
                break;
            case 2: // Calculate cos(x)
                printf("\n%g", cos(x * PI / 180)); // Convert x to radians and compute cos(x)
                break;
            case 3: // Calculate tan(x)
                printf("\n%g", tan(x * PI / 180)); // Convert x to radians and compute tan(x)
                break;
            case 4: // Calculate natural log of x
                printf("\n%g", log(x)); // Compute natural logarithm of x
                break;
            case 5: // Calculate base-10 log of x
                printf("\n%g", log10(x)); // Compute base-10 logarithm of x
                break;
            case 6: // Calculate e^x
                printf("\n%g", exp(x)); // Compute e raised to the power x
                break;
            case 7: // Calculate square root of x
                printf("\n%g", sqrt(x)); // Compute square root of x
                break;
            case 8: // Calculate factorial of x
                for (f = 1, i = 2; i <= x; f *= i, i++); // Compute factorial iteratively
                printf("\n%ld", f); // Print the factorial result
                break;
            default:
                continue; // Continue to the next iteration for invalid choices
        }

        getch(); // Wait for a key press to continue

    } while (1); // Repeat the loop indefinitely

    return 0; // Exit the program
}
```
